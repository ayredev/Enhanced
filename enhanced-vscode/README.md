# Enhanced Language Support for VSCode

Official syntax highlighting and language configuration for the **Enhanced** programming language.

## Features
- Syntax highlighting for all verbs, nouns, and structural phrases.
- Custom bracket matching and auto-closing string pairs.
- Smart auto-indentation inside loops and condition blocks.

## Usage
Simply install this `.vsix` package in Visual Studio Code. Files ending in `.en` will automatically be recognized as Enhanced source code!
